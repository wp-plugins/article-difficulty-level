Select the post difficulty level by rating it.

== Description ==
Through this plugin, user can able to select the post difficulty level in admin dash board post area for each post.

== Installation ==

1. Upload the "article-difficulty-level" directory to the plugins directory.
2. Go to the plugins setting page and activate "Post Difficulty Level". 
3. Users can see the Post Strenght mode in Posting area.
